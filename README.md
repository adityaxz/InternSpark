# InternSpark AI/ML Internship Tasks

This repository contains the completed tasks submitted for the InternSpark AI/ML Internship Program.

## Tasks Completed

### Task 1 – Machine Learning Classification

Developed and evaluated machine learning models for breast cancer classification using the Breast Cancer Wisconsin dataset.

**Files:**

* 1mlclassification.ipynb
* 1report.md
* Supporting screenshots

**Techniques Used:**

* Exploratory Data Analysis (EDA)
* Logistic Regression
* Random Forest Classifier
* Model Evaluation
* Cross Validation

---

### Task 3 – Model Deployment: API & Containerization

Deployed a trained machine learning model as a REST API using Flask and prepared it for containerized deployment using Docker.

**Files:**

* 3apiintegration.ipynb
* 3app.py
* Dockerfile
* 3report.md
* breast_cancer_model.pkl
* Supporting screenshots

**Features:**

* REST API endpoint for predictions
* JSON request/response handling
* Docker configuration
* Local API testing

---

### Task 4 – Responsible AI & Model Interpretation

Performed model interpretability and fairness analysis using feature importance and SHAP explainability techniques.

**Files:**

* 4responsibleai.ipynb
* 4report.md
* Supporting screenshots

**Techniques Used:**

* Feature Importance Analysis
* SHAP Explainability
* Fairness Assessment
* Responsible AI Recommendations
